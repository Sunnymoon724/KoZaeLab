# 코재의 개인적인 코드 연구소

코 코재가 범용적으로 코드를 쓸 수 있도록 라이브러리화 시키고 있는 연구소. (현재 코드 정리 중)

제대로 사용하기 위해서는 **DotweenPro**, **Odin Inspector** 에셋이 필요하며 **UniTask** 패키지를 사용하고 있습니다.

PlayFab 같은 경우는 **KZPlayFab**을 사용시 필요합니다.
